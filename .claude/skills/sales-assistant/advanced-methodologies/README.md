# Advanced Methodologies — продвинутые фреймворки

Расширения базовой методологии Софии Добрыниной (которая лежит в корне скилла) для сложных кейсов.

## Файлы

### 🎯 [KVP_MASTER_SCRIPT.md](./KVP_MASTER_SCRIPT.md)
Внутренний регламент отдела продаж AI-Ковчега (КВП). 5 принципов перед звонком + 8-блочная структура: Привітання → Афініті → Анамнез (SPIN) → Резюме → Презентация → Цена → Дожим → Закрытие.

**Когда использовать:** хай-тикет продукты ($500+), 1-на-1 звонки, длинные циклы сделки. Особенно хорошо для образовательных и консалтинговых продуктов.

### 🎤 [MULTI_VOICE_FRAMEWORK.md](./MULTI_VOICE_FRAMEWORK.md)
Фреймворк работы с несколькими голосами в одной команде продаж. Кейс-разбор: 2 менеджера AI-Ковчега (Анастасия эмпатичная / Владислав деловой) — как анализировать ToV каждого, когда какой выбирать, готовые шаблонные фразы, decision tree выбора голоса под клиента.

**Когда использовать:** когда в твоей команде 2+ менеджера с разными стилями. Ассистент сможет адаптироваться под голос каждого.

## Связь с базовой методологией

Эти файлы **дополняют**, а не заменяют базу:
- `01_МЕТОДОЛОГИЯ_ПРОДАЖ.md` — общая философия продаж в переписке
- `02_ПРАВИЛА_ОБЩЕНИЯ.md` — правила структуры сообщений
- `03_УСПЕШНЫЕ_ПЕРЕПИСКИ.md` — 4 универсальных кейса

КВП-скрипт берёт SPIN из базы и упаковывает в более жёсткую структуру 8 блоков. Multi-voice добавляет адаптацию ToV поверх формулы Цель→Вопросы→Решение→Действие.
